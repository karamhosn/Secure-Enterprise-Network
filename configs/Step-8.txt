8. OSPF on L3 switches, ISP routers and INTERNET/CLOUD router. 

To see OSPF on CORE-SW1 enter command (do) show start and click more until you see something like:

!
router ospf 35
 router-id 1.1.1.1
 log-adjacency-changes
 network 10.2.2.0 0.0.0.3 area 0
 network 10.2.2.4 0.0.0.3 area 0
 network 192.168.10.0 0.0.0.255 area 0
 network 172.16.0.0 0.0.255.255 area 0
 network 10.20.0.0 0.0.255.255 area 0
 network 10.11.11.32 0.0.0.31 area 0
!


To see OSPF on CORE-SW1 enter command (do) show start and click more until you see something like:

router ospf 35

 network 172.16.0.0 0.0.255.255 area 0
 network 10.20.0.0 0.0.255.255 area 0
 network 10.11.11.32 0.0.0.31 area 0
!
